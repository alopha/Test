# Privacy Policy for Baidu Search Tool

## Data Collection
The Baidu Search Tool collects search queries that you provide to perform searches on Baidu through SerpAPI's service. This includes the search terms and parameters you specify when using the tool.

## Data Processing
- Search queries are sent to SerpAPI's servers to retrieve search results from Baidu
- The tool processes and formats the search results before returning them to you
- No search queries or results are stored permanently by this tool

## Third-party Services
This tool uses the following third-party services:
- **SerpAPI**: Handles the actual search requests to Baidu. Please refer to [SerpAPI's Privacy Policy](https://serpapi.com/privacy) for information about how they handle your data.
- **Baidu**: The search engine that provides the actual search results. Please refer to [Baidu's Privacy Policy](https://privacy.baidu.com/) for information about how they handle your data.

## Data Retention
- Search queries are not stored by this tool after the search is completed
- Search results are not cached or stored permanently
- Your SerpAPI key is stored securely for authentication purposes only

## User Rights
You have the right to:
- Stop using the tool at any time
- Contact us with privacy-related concerns
- Request information about what data is collected (if any)
